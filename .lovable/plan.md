

## Aggiornamento API Key di Resend

### Cosa faremo

Aggiorneremo il segreto `RESEND_API_KEY` con la nuova chiave API che hai copiato dalla dashboard di Resend.

### Passaggi

1. Aggiornare il segreto `RESEND_API_KEY` con il nuovo valore
2. Testare la funzione con una chiamata di prova per verificare che l'errore "API key is invalid" sia risolto

### Risultato atteso

Le email di richiesta preventivo verranno inviate correttamente sia al business che al cliente.
